# Goのインストール

1. ターミナルを開く

2. brewのインストール
    ```bash
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    ```

3. brewのバージョン確認
    ```bash
    brew --version
    ```

4. Goのインストール
    ```bash
    brew install go
    ```

5. Goのインストール確認
    ```bash
    go version
    ```
